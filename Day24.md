
# День 24 — Финальная сборка прототипа

⏱ Время выполнения: **30–45 минут**

---

# 🎯 Цель дня

Проверить все системы игры и собрать **играбельный прототип**.

После этого дня игра будет иметь:

```
волны врагов
башни
экономику
жизни
HUD
экраны победы и поражения
продажу башен
```

Это называется **MVP (Minimum Viable Product)**.

---

# 🧩 Архитектура всей игры

```
WaveSpawner
↓
Enemies spawn
↓
EnemyMovement
↓
Башни атакуют
↓

↓

↓

```

---

# 🔧 Шаг 1 — Проверить спавн врагов

Нажми **Play**.

Проверь:

```
враги появляются
враги движутся по пути
```

---

# 🔧 Шаг 2 — Проверить стрельбу башен

Проверь:

```
башни находят врагов
башни стреляют
враги получают урон
```

---

# 🔧 Шаг 3 — Проверить экономику

Проверь:

```
убийство врага даёт золото
строительство башни тратит золото
продажа башни возвращает золото
```

---

# 🔧 Шаг 4 — Проверить систему жизней

Пропусти врага.

Проверь:

```
Lives уменьшается
```

---

# 🔧 Шаг 5 — Проверить Game Over

Пропусти достаточно врагов.

Результат:

```
Game Over Panel появляется
игра останавливается
```

---

# 🔧 Шаг 6 — Проверить Victory

Пройди все волны.

Результат:

```
Victory Panel появляется
игра останавливается
```

---

# 🔧 Шаг 7 — Проверить консоль

Открой:

```
Window → General → Console
```

Проверь:

```
нет красных ошибок
```

---

# 🔧 Шаг 8 — Подготовить сборку

Открой:

```
File
Build Settings
```

---

# 🔧 Шаг 9 — Добавить сцену

Нажми:

```
Add Open Scenes
```

В списке должна появиться:

```
Main.unity
```

---

# 🔧 Шаг 10 — Выбрать платформу

```
Platform
PC, Mac & Linux Standalone
```

---

# 🔧 Шаг 11 — Сделать билд

Нажми:

```
Build
```

Выбери папку:

```
Build/
```

Unity создаст файл:

```
Game.exe
```

---

# ✅ Проверка билда

Запусти:

```
Game.exe
```

Проверь:

```
игра запускается
враги появляются
башни строятся
волны работают
```

---

# ⚠️ Частые ошибки

### ❌ Build не запускается

Причина:

```
сцена не добавлена в Build Settings
```

---

### ❌ Чёрный экран

Причина:

```
нет активной камеры
```

---

### ❌ Башни не стреляют

Причина:

```
TowerController не видит врагов
```

Проверь:

```
Enemy Tag = "Enemy"
```
