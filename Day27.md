
# День 27 — Босс (Fire Golem) + визуальные эффекты появления

⏱ Время выполнения: **20–30 минут**

---

# 🎯 Цель дня

Создать мощного босса с эпичным появлением.  
Босс будет большим, медленным и очень сильным — последняя волна должна пугать игрока!

---

# 🧩 Что будет добавлено

- Новый префаб **EnemyBoss**
- Particle System (огонь вокруг босса)
- Скрипт **BossEnemy** с тряской камеры

---

# 🔧 Шаг 1 — Создаём префаб босса

1. В окне **Project** открой папку:
   ```
   Assets → Prefabs → Enemies
   ```
2. Выбери любой готовый префаб (например **EnemyTroll.prefab**)
3. Правой кнопкой мыши → **Duplicate**
4. Переименуй новый файл в **EnemyBoss.prefab**
5. Двойной клик — открой префаб для редактирования

---

# 🔧 Шаг 2 — Меняем внешний вид и характеристики босса

В Inspector измени:

- **Sprite Renderer** → Sprite → выбери большой огненный голем (из Kenney или своего спрайтшита)
- **Transform** → Scale → **2.2 / 2.2 / 1**
- **Circle Collider 2D** → Radius → **1.2**
- **EnemyHealth** → Max Health → **2200**, Gold Reward → **120**
- **EnemyMovement** → Speed → **1.1** (очень медленный)

**Зачем такие значения?**  
Босс должен быть огромным, тяжёлым и давать много золота — игрок будет чувствовать, что это настоящий финальный враг.

---

# 🔧 Шаг 3 — Добавляем огонь вокруг босса

1. На префабе босса нажми **Add Component** → найди **Particle System**
2. В Inspector настрой Particle System:
   - Rate over Time → **20**
   - Start Lifetime → **0.8**
   - Start Size → **0.3**
   - Color → ярко-оранжевый/красный
   - В разделе **Shape** → Shape → **Circle**

**Зачем Particle System?**  
Это готовый инструмент Unity для красивых эффектов. Огонь вокруг босса сразу показывает, что это опасный враг.

---

# 🔧 Шаг 4 — Создаём скрипт BossEnemy

1. В папке **Assets/Scripts**:
   - Правой кнопкой → **Create** → **C# Script**
   - Назови **BossEnemy**
2. Открой скрипт и полностью замени код на:

```csharp
using UnityEngine;

public class BossEnemy : MonoBehaviour
{
    private void Start()
    {
        // Экранная тряска при появлении
        if (Camera.main.TryGetComponent<CinemachineImpulseSource>(out var impulse))
        {
            impulse.GenerateImpulse(Vector3.one * 0.8f);
        }
    }
}
```

---

# 🔧 Шаг 5 — Прикрепляем скрипт и добавляем тряску камеры

1. На префабе **EnemyBoss** нажми **Add Component** → найди **BossEnemy**
2. (Опционально, но очень круто) На **Main Camera**:
   - Add Component → **Cinemachine Impulse Source**
   - Оставь настройки по умолчанию

**Зачем тряска?**  
Когда босс появляется — камера трясётся! Это сразу вызывает эмоцию «о нет, босс!» — как в профессиональных играх.

# ✅ Проверка дня 27

1. Перетащи **EnemyBoss.prefab** в сцену
2. Нажми **Play**

Что должно быть:
- Босс идёт очень медленно
- Много здоровья (2200)
- Вокруг него горит огонь
- При появлении камера слегка трясётся

---

# ⚠️ Частые ошибки

### ❌ Нет тряски камеры
Причина: Cinemachine Impulse Source не добавлен на Main Camera  
Решение: добавь компонент на камеру

### ❌ Босс слишком маленький
Причина: Scale не поменяли  
Решение: поставь 2.2 × 2.2

### ❌ Нет частиц огня
Причина: Particle System не настроен  
Решение: проверь Rate over Time = 20
