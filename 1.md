### День 1 — Создание проекта и базовая настройка сцены

1. Запусти **Unity Hub**  
   - Ищи значок с буквой U (на рабочем столе, в меню Пуск или поиске Windows).  
   - Дважды кликни — откроется окно Unity Hub.

2. Нажми **New project**  
   - Большая синяя кнопка справа сверху (или с плюсом).

3. Выбери шаблон 2D  
   - Слева в списке шаблонов найди раздел **Core** или **2D**.  
   - Выбери **2D** или **Universal 2D** (самый простой для 2D-игры).  
   - Если написано **Download** — нажми, подожди 1–5 минут (скачивается один раз).

4. Назови проект  
   - В поле **Project name** напиши: `TowerDefence2026` (или любое имя без пробелов).  

5. Выбери папку для проекта  
   - Нажми **…** рядом с Location.  
   - Перейди на диск D: (или C:).  
   - Правой кнопкой → **New** → **Folder** → назови `UnityProjects`.  
   - Открой эту папку → нажми **Select Folder**.  
   - Полный путь будет примерно: `D:\UnityProjects\TowerDefence2026`.

6. Создай проект  
   - Нажми большую синюю кнопку **Create project** внизу справа.  
   - Подожди 1–3 минуты (первый запуск дольше) — откроется редактор Unity.

**Что ты видишь сразу после открытия?**  
- Слева — **Hierarchy** (список объектов сцены): Main Camera и Directional Light.  
- В центре — **Scene** окно (серое поле).  
- Справа — **Inspector** (настройки).  
- Внизу — **Project** (папки Assets) и **Console**.

Готово! Проект создан.

### После открытия проекта — чистим и настраиваем первую сцену

#### 8. Находим и удаляем ненужную сцену SampleScene.unity

**Где лежит окно Project?**  
- Самая нижняя панель экрана.  
- Если не видно — кликни по вкладке **Project** внизу (рядом Console, Game и т.д.).  
- Или нажми **Ctrl + 9**.

**Что внутри Project?**  
- Раскрой **Assets** (стрелка слева).  
- Внутри Assets ищи папку **Scenes** → раскрой её.  
- Там будет файл **SampleScene.unity** (иконка — синий кубик с U).

**Как удалить:**  
1. Кликни по **SampleScene.unity** левой кнопкой (подсветится).  
2. Правой кнопкой → **Delete**.  
3. В окошке подтверждения нажми **Delete** (или Yes).

**Почему удаляем именно сейчас?**  
- Unity по умолчанию загружает последнюю сцену при запуске.  
- Если оставить SampleScene — каждый Play будет показывать старую тестовую сцену с кубиком.  
- При сборке игры (Build) она попадёт в финальный билд → игрок увидит кубик вместо твоей карты.  
- Удаление = чистый проект с самого начала.

---

#### 9. Создаём новую чистую сцену

1. В верхнем меню нажми **File** → **New Scene**.  
2. Появится окошко **Create New Scene** (или **New Scene Template**).  
3. Выбери шаблон **Basic (Built-in)** — самый первый или второй в списке.  
   - Не бери URP, HDRP, 2D Core — они добавляют лишние настройки.  
4. Нажми **Create**.

**Что будет после Create?**  
- Hierarchy очистится.  
- Останутся только: **Main Camera** и **Directional Light**.  
- Scene окно станет серым/белым — пустая сцена.

**Почему именно Basic (Built-in)?**  
- Минимум объектов и настроек.  
- Нет URP/HDRP, пост-эффектов, лишних камер — ничего, что запутает новичка.  
- Идеально для 2D Tower Defense с ортографической камерой.

---

#### 10. Сразу сохраняем сцену как Main.unity

1. Нажми **File** → **Save Scene As…** (или **Ctrl + Shift + S**).  
2. В окне сохранения слева выбери **Assets**.  
3. Если внутри Assets уже есть **Scenes** — открой её.  
   Если нет: правой кнопкой в Assets → **Create** → **Folder** → назови **Scenes** → открой.  
4. В поле **File name** внизу напиши **Main**.  
5. Нажми **Save**.

**Почему важно сохранить сразу?**  
- Без сохранения сцена живёт только в памяти. Закроешь Unity — всё пропадёт.  
- Название **Main.unity** сразу понятно: это главная игровая сцена.  
- Путь будет: `Assets/Scenes/Main.unity`.

---

#### 11. Настраиваем камеру под 2D

1. В Hierarchy кликни на **Main Camera**.  
2. В Inspector (справа) измени три поля:

   - **Projection**  
     Кликни стрелку вниз рядом с Perspective → выбери **Orthographic**.  
     **Зачем?** 2D-игры не используют перспективу — всё должно быть плоским и без искажений.

   - **Size**  
     Сотри текущее число → напиши **5.5** → Enter.  
     **Зачем?** Это высота видимой области. 5.5 — удобно для карты ~10–12 клеток в высоту.

   - **Background**  
     Кликни на цветной квадратик.  
     Внизу в поле **Hex Color** напиши **1A3C1A** (без #) → Enter.  
     **Зачем?** Тёмно-зелёный фон — классика для травы/карты, глаза не устают.

---

#### 12. Финальное сохранение

Нажми **Ctrl + S** или **File → Save**.

**Проверка дня 1**  
- Нажми **Play** (треугольник сверху).  
- Экран должен стать тёмно-зелёным и пустым.  
- Нажми Play снова — остановка.  
- Всё работает → день пройден.

**Итог дня 1**  
- Чистый проект без мусора.  
- Сцена Main.unity с настроенной 2D-камерой.  
- Готовы к тайлам и карте (День 2).
