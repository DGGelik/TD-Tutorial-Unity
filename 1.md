# План разработки: Tower Defence  
**Детальный план по шагам — Месяц 1**

**Цель месяца:** Рабочий прототип с движущимися врагами и одной башней, которая стреляет.  
**Результат к концу месяца:** Можно запустить игру, увидеть врагов, идущих по пути, и башню, которая их уничтожает.  
**Время на месяц:** 8–12 занятий по 1,5–2 часа + домашние задания.

## [Неделя 1: Макет уровня и путь](Day1-7.md)



### День 1: Создание проекта и базовая сцена
**Что делаем:**
```
1. File → New Project → 2D (Core)
2. Удалить Sample Scene → File → New Scene → Basic (2D) → MainScene
3. Настройка камеры:
   - Main Camera → Projection: Orthographic
   - Size: 8–10 (видно всю карту)
   - Background: тёмно-зелёный (#2A4A2A)
4. Window → Package Manager → 2D Tilemap Extras → Install
5. Сохранить сцену: Assets/Scenes/MainScene.unity
```

**Результат:** Пустая сцена с настроенной камерой.

### День 2: Создание Tilemap и рисование карты
**Что делаем:**
```
1. Hierarchy → Create Empty → "Grid" (Scale: 1,1,1)
2. Grid → Create Empty → Tilemap → "GroundTilemap" (трава)
3. Grid → Create Empty → Tilemap → "RoadTilemap" (дорога)
4. Window → 2D → Tile Palette
5. Импорт ассетов Kenney Tower Defense:
   - Скачать с kenney.nl → Assets/Tilesets/
   - Перетащить траву → Tile Palette → Create New Palette → Ground
   - Перетащить дорогу → Road Palette
6. Рисуем карту 40x30 тайлов:
   - GroundTilemap: заполнить всю карту травой
   - RoadTilemap: змейка длиной 130–150 тайлов (12–15 поворотов)
```

**Результат:** Красивая карта с извилистой дорогой.

### День 3: Waypoints для пути врагов
**Что делаем:**
```
1. Create Empty → "Waypoints" (родитель)
2. Создать 16 пустых GameObject: Waypoint0 → Waypoint15
3. Расставить точно по центру каждого сегмента дороги
4. Waypoints → Add Component → Context → "Make Waypoints Child Of Grid"
5. Создать ScriptableObject для пути:
   - Assets → Create → ScriptableObject → WaypointPath.asset
   - public Transform[] waypoints = new Transform[16];
```

**Результат:** Враги смогут идти точно по дороге.

**Контрольная точка недели 1:** Камера показывает полную карту с дорогой и 16 waypoints ✅

## Неделя 2: Система движения врагов (waypoints)

### День 4: Первый враг — префаб Slime
**Что делаем:**
```
1. Create Empty → "EnemySlime" (Scale: 0.8,0.8,1)
2. SpriteRenderer → спрайт slime из Kenney
3. Add Component → CircleCollider2D (IsTrigger: true)
4. Add Component → Rigidbody2D → Body Type: Kinematic
5. Сохранить как префаб: Assets/Prefabs/Enemies/EnemySlime.prefab
```

**Результат:** Готовый враг-префаб.

### День 5: Скрипт движения врага
**Что делаем:**
```
1. Assets → Create → C# Script → EnemyMovement
2. Прикрепить к EnemySlime
3. Основная логика:
   - public Transform[] waypoints
   - public float moveSpeed = 2f
   - int waypointIndex = 0
   - Движение к текущей точке
   - Переход к следующей при достижении
4. Назначить WaypointPath.waypoints в инспекторе
```

**Результат:** Враг самостоятельно идёт по всему пути.

### День 6: Здоровье врага и смерть
**Что делаем:**
```
1. Assets → Create → ScriptableObject → EnemyStats_Slime
   - public float maxHealth = 100f
   - public float goldReward = 6f
2. EnemySlime → Add Component → EnemyHealth
3. Логика: TakeDamage(), Death() → Destroy(gameObject)
4. Добавить Canvas → World Space → HP бар над врагом
```

**Результат:** Враг умирает при получении урона.

**Контрольная точка недели 2:** Один враг спавнится и самостоятельно проходит весь путь ✅

## Неделя 3: Базовый спавнер волн

### День 7: Wave Spawner
**Что делаем:**
```
1. Create Empty → "GameManager"
2. Add Component → WaveSpawner
3. Логика:
   - public Transform spawnPoint (Waypoint0)
   - public GameObject enemyPrefab
   - float timeBetweenWaves = 5f
   - UI countdown между волнами
4. Canvas → Text → "Wave Incoming: 3... 2... 1..."
```

**Результат:** Волны спавнятся автоматически.

### День 8: Простая первая волна
**Что делаем:**
```
WaveSpawner:
- Волна 1: 5 слизней с интервалом 0.8 сек
- После волны: пауза 10 сек + следующая
- Максимум 3 волны для теста
```

**Результат:** 3 волны по 5 врагов.

**Контрольная точка недели 3:** Волны спавнятся автоматически, враги идут по пути ✅

## Неделя 4: 1 тип башни + снаряд

### День 9: Archer Tower префаб
**Что делаем:**
```
1. Create Empty → "ArcherTower" (Scale: 1.2,1.2,1)
2. SpriteRenderer → башня из Kenney
3. CircleCollider2D (IsTrigger, Radius: 3.5)
4. Пустой дочерний объект "Turret" (поворот башни)
5. Префаб: Assets/Prefabs/Towers/ArcherTower.prefab
```

### День 10: Скрипт башни (поиск цели + стрельба)
**Что делаем:**
```
ArcherTower → TowerController:
- float range = 3.5f, damage = 25f, fireRate = 1.2f
- Enemy target (ближайший в радиусе)
- Update(): поворот турели + стрельба
```

### День 11: Снаряд (Bullet)
**Что делаем:**
```
1. Префаб Bullet: маленький спрайт + Rigidbody2D
2. BulletController:
   - public Transform target
   - Гоминг движение к цели
   - OnTriggerEnter2D → урон + Destroy
3. TowerController: Instantiate(bulletPrefab, spawnPoint, Quaternion.identity)
```

### День 12: Простое размещение башен
**Что делаем:**
```
1. Canvas → 4 кнопки башен (пока только Archer)
2. BuildManager:
   - Raycast2D от мыши
   - Если трава + хватает золото → Instantiate башню
3. Стартовое золото: 200
```

**Контрольная точка месяца 1:** Полностью рабочий прототип! ✅
```
- Враги идут волнами по пути
- Ставим башни мышкой за золото
- Башни поворачиваются и стреляют пулями
- Враги умирают от урона
- Золото за убийства
```
