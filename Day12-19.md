# 4. Что такое Prefab

Prefab — это **шаблон объекта**.

Например башня:

```
ArcherTower
```

Prefab хранит:

* модель или спрайт
* коллайдер
* скрипты
* настройки

Когда мы создаём башню в игре:

```
Instantiate(ArcherTower)
```

Unity **копирует prefab**.

# День 14 — Система данных башен (ScriptableObject)

---

# 1. Проблема текущей системы

Сейчас параметры башни могут храниться:

* в скрипте
* в prefab

Это плохо.

Почему:

Если появится **20 башен**:

вам придётся открывать **20 prefab** чтобы изменить баланс.

---

# 2. Решение

Используем:

```
ScriptableObject
```

Это файл данных.

Он хранит:

```
damage
fireRate
range
cost
```

Башни будут **читать данные из него**.

Это стандарт **Unity архитектуры**.

---

# 3. Создаём скрипт

Перейдите в папку:

```

```

```

```


---

# 4. Код TowerData

Полностью замените код:

```csharp
using UnityEngine;

[CreateAssetMenu(fileName = "TowerData", menuName = "TowerDefense/TowerData")]
public class TowerData : ScriptableObject
{
    [Header("Basic Info")]
    public string towerName = "Tower";

    [Header("Base Stats")]
    public float baseDamage = 20f;
    public float baseFireRate = 1f;
    public float baseRange = 3.5f;
    public int baseCost = 80;

    [Header("Upgrade Costs")]
    public int[] upgradeCosts =
    {
        120,
        180
    };

    [Header("Damage Multipliers")]
    public float[] damageMultiplier =
    {
        1f,
        1.5f,
        2f
    };

    [Header("FireRate Multipliers")]
    public float[] fireRateMultiplier =
    {
        1f,
        1.3f,
        1.6f
    };

    [Header("Range Multipliers")]
    public float[] rangeMultiplier =
    {
        1f,
        1.2f,
        1.4f
    };

    [Header("Visual")]
    public Sprite icon;
    public GameObject towerPrefab;
}
```

# День 17 — Ghost башни

---

Ghost — полупрозрачная башня.

Она показывает **где будет стоять башня**.

---

# Код

```csharp
GameObject ghost;

void Update()
{
    if(selectedTower != null)
    {
        Vector3 pos = Camera.main.ScreenToWorldPoint(Input.mousePosition);
        pos.z = 0;

        if(ghost == null)
        {
            ghost = Instantiate(selectedTower.towerPrefab);
        }

        ghost.transform.position = pos;
    }
}
```
