
# День 25 — Префабы остальных врагов (Spider, Troll, Wraith)

---

# 🔧 Шаг 1 — Подготовка папки

1. В окне **Project** открой папку:
   ```
   Assets → Prefabs → Enemies
   ```
2. Если папки **Enemies** нет — создай её:
   - Правой кнопкой мыши на папке **Prefabs** → **Create** → **Folder** → назови **Enemies**

**Зачем отдельная папка?**  
Чтобы всё было аккуратно. В реальных проектах все префабы врагов лежат в одной папке — так проще искать.

---

# 🔧 Шаг 2 — Создаём префаб паука

1. В папке **Enemies** найди **EnemySlime.prefab**
2. Правой кнопкой мыши → **Duplicate**
3. Переименуй новый файл в **EnemySpider.prefab**
4. Двойной клик по **EnemySpider.prefab** (откроется режим редактирования префаба)

---

# 🔧 Шаг 3 — Меняем внешний вид и характеристики паука

В Inspector измени:

- **Sprite Renderer** → Sprite → выбери спрайт паука (из Kenney или своего спрайтшита)
- **Transform** → Scale → **0.9 / 0.9 / 1**
- **Circle Collider 2D** → Radius → **0.45**
- **EnemyHealth** → Max Health → **180**, Gold Reward → **12**
- **EnemyMovement** → Speed → **3.2** (быстрее слизня)

**Зачем разные значения?**  
Паук должен быть быстрым и слабым — игроку придётся ставить больше башен, чтобы успеть его убить.

---

# 🔧 Шаг 4 — Создаём тролля и призрака (точно так же)

Повтори шаги 2–3 для двух других префабов:

**EnemyTroll.prefab**
- Sprite → тролль/каменный голем
- Scale → **1.4 / 1.4 / 1**
- Collider Radius → **0.7**
- EnemyHealth → Max Health **550**, Gold Reward **30**
- EnemyMovement → Speed **1.4** (медленный и тяжёлый)

**EnemyWraith.prefab**
- Sprite → призрак/дух
- Scale → **1.0 / 1.0 / 1**
- Collider Radius → **0.5**
- EnemyHealth → Max Health **280**, Gold Reward **18**
- EnemyMovement → Speed **2.8**

# ✅ Проверка дня 25

1. Перетащи все 4 префаба (Slime, Spider, Troll, Wraith) в сцену
2. Нажми **Play**

Что должно быть:

---

# ⚠️ Частые ошибки

### ❌ Враг выглядит как слизень
Причина: не поменяли Sprite в Sprite Renderer  
Решение: открой префаб и выбери правильный спрайт

### ❌ Все враги имеют одинаковую скорость
Причина: забыли поменять Speed в EnemyMovement  
Решение: проверь каждый префаб отдельно

### ❌ Золото не даётся за убийство
Причина: в EnemyHealth нет поля Gold Reward (из Дня 20)  
Решение: добавь поле и заполни
