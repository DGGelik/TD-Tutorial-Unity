Ниже — **практическая таблица настроек Particle System** для типичных эффектов в Tower Defense. Значения ориентированы на **Unity 2D/2.5D**, короткие burst-эффекты (0.2–1.5 сек), чтобы не перегружать сцену. Используй их как **стартовые пресеты**, затем подстраивай под стиль игры.

> Общие рекомендации:
> • `Simulation Space = World` для ударов/взрывов, `Local` — для эффектов, привязанных к объекту.
> • `Looping = Off` для разовых эффектов.
> • `Stop Action = Destroy` (или `Disable`, если используешь пул).
> • `Max Particles` держи в пределах 50–300.

## Быстрые настройки материалов (важно для “свечения”)

Для ярких эффектов (огонь, магия, молния):

| Параметр       | Значение                     |
| -------------- | ---------------------------- |
| Shader         | **Particles/Standard Unlit** |
| Rendering Mode | Additive                     |
| Texture        | Soft circle / spark          |
| Color          | Яркий (HDR если URP + Bloom) |

Это даст **неоновый glow-эффект**.

## Рекомендованные лимиты для производительности

| Тип эффекта    | Частицы     |
| -------------- | ----------- |
| Hit            | 10–20       |
| Fire explosion | 40–70       |
| Enemy death    | 40–80       |
| Build effect   | 40–60       |
| DOT эффекты    | 10–25 / sec |

Старайся держать **< 200 активных частиц одновременно** на одном эффекте.

---

## Полезные модули Particle System

| Модуль                 | Для чего использовать    |
| ---------------------- | ------------------------ |
| Color over Lifetime    | постепенное исчезновение |
| Size over Lifetime     | частицы “сжимаются”      |
| Velocity over Lifetime | движение вверх/в сторону |
| Noise                  | хаотичность (дым, магия) |
| Trails                 | следы (молния, искры)    |
| Light                  | вспышки (огонь, магия)   |

## Практический совет

Лучше сделать структуру префабов:

```
Assets
   Effects
      Build
      Fire
      Ice
      Lightning
      Magic
      Death
```

И каждый эффект хранить как **Prefab**, чтобы вызывать его из кода.
